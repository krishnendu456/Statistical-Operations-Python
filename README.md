# Statistical Operations in Python

## Overview

This project demonstrates the implementation of basic statistical operations in Python using the built-in `statistics` module. It is intended for beginners to understand fundamental statistical concepts and their implementation in Python.

## Features

* Calculation of mean
* Calculation of median
* Calculation of mode
* Variance computation
* Standard deviation calculation
* Exception handling for datasets without a unique mode

## Concepts Covered

* Descriptive statistics
* Python standard library (`statistics`)
* Exception handling

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:

```
git clone https://github.com/krishnendu456/Statistical-Operations-Python.git
```

2. Navigate to the project directory:

```
cd Statistical-Operations-Python
```

3. Run the program:

```
python stats_program.py
```
